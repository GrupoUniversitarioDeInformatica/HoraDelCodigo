# STRUDEL - Strudel: Musica y programacion

- Autor: Mangel
- Fecha de adición: 2025-11-12 17:21:07

---

## Descripción

Este taller consiste en aprender las bases musicales suficientes como para poder componer de forma sencilla música programando en la plataforma [Strudel](strudel.cc)

Este taller no tiene materiales, todos los contenidos saldrán de lo desarrollado durante el taller.

## Requisitos

Se recomienda a los participantes que:

- Venir con portátil y Windows instalado en él
- Altamente recomendable venir con auriculares

## Otros detalles

Para el que quiera dar este taller, recomiendo que tenga los suficientes conocimientos musicales sobre producción musical como para dar el taller. Teoría musical, acordes, ritmos, melodías, filtros, frecuencias, etc...
