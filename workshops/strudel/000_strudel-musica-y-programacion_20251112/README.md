## Descripción

Este taller va de crear una serie de ritmos, melodías y otros asuntos musicales a través de la programación.

## Requisitos

Se recomienda a los participantes que:

- Traiga portátil con Windows instalado
- Auriculares para evitar molestar a otros asistentes

## Otros detalles

Para el que quiera dar este taller, recomiendo que tenga control con la producción musical, debe saber crear ritmos, melodías, acordes, filtros, etc...
