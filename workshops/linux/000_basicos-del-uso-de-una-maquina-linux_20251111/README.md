Este taller va de como usar una máquina Linux con la suficiente destreza como para que no tengas problemas a la hora de trabajar en una máquina con este sistema operativo sin mayores contratiempos. Además, este taller será muy recomendable para estudiantes de Telecomunicaciones e Informática, que van a trabajar muchas veces sobre máquinas con estos sistemas operativos.

## Requisitos

Se recomienda a los participantes que:

- Traigan ordenador propio con Linux instalado (pueden haberlo instalado anteriormente en la Install Party), en su defecto que se aseguren que en el taller en el que se imparte el taller tenga puestos en los que pueda acceder a Linux
- Tengan ganas de aprender
