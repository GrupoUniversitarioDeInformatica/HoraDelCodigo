## Descripción

Este taller va de aprender a escribir tests para código en Python, los diferentes tipos de tests que existen y las librerías más populares y frecuentes para trastear código antes de que toque producción.

- Librerías
  - unittest
  - pytest
  - hypothesis
  - testcontainers
  - fastapi test clients
  - pytest-cov
  - pytest-xdist
  - pytest-asyncio
  - ...

- Tipos de tests
  - Unit
  - Regression
  - Integration
  - End-2-end
  - Smoke
  - ...

- Metodologías
  - Test Driven Development
    - Ping Pong Programming

- Conceptos básicos
  - Mocking
  - Fixtures
  - Parameterized
  - ipdb --> breakpoints para depuración del código
  - marks (skip, skipif)

## Requisitos

Se recomienda a los participantes que:

- tengan un conocimiento medio programando en Python
- traigan sus propios portátiles

## Otros detalles

Para el que quiera dar este taller, recomiendo que tenga experiencia real haciendo tests en Python.

Todo el código está ya preparado para los tests, para ilustrar cada concepto y librerías mencionadas arriba.

Leed el fichero [WORKSHOP_GUIDE.md](./WORKSHOP_GUIDE.md) para dar este taller.
